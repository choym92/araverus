# 2.2 News SEO

## Overview

SEO configuration for araverus.com тАФ metadata, structured data, sitemap, RSS, and robots strategy. Primary focus: maximizing Google discoverability of `/news` content. All metadata, OG, Twitter, and JSON-LD rebranded to Araverus Organization.

## Site-wide (`src/app/layout.tsx`)

### Metadata
- **title**: `{ default: "Araverus тАФ Financial Intelligence", template: "%s | Araverus" }`
- **description**: `"Financial intelligence platform powered by AI, machine learning, and neural networks."`
- **metadataBase**: `https://araverus.com` тАФ resolves relative URLs in openGraph/twitter
- **openGraph**: type `website`, locale `en_US`, siteName `araverus.com`, image `og-news-default.png` (1200x630)
- **twitter**: `summary_large_image` card, same image
- **alternates**: RSS link at `/rss.xml`, Podcast RSS link at `/podcast.xml`

### JSON-LD (root `<body>`)
Two schemas injected on every page:

```json
[
  {
    "@type": "WebSite",
    "name": "Araverus",
    "url": "https://araverus.com",
    "description": "Financial intelligence platform powered by AI, machine learning, and neural networks."

    "@type": "Organization",

    "url": "https://araverus.com",

```

**Purpose**: Establish Araverus as an Organization brand identity in Google Knowledge Graph.

### Fonts
All three fonts use `display: 'swap'` to prevent invisible text during load (CLS optimization):
- Inter (body)
- Geist Mono (code)
- Playfair Display (serif headings)

## `/news` page (`src/app/news/page.tsx`)

### Metadata
- **title**: `{ absolute: titleText }` тАФ bypasses `| Araverus` template
  - Default: `"AI News Briefing тАФ Tech, Markets & Finance"`
  - Per category: `"${label} News тАФ AI Briefing"` (e.g. `"Tech News тАФ AI Briefing"`)
- **description**: threading-focused, dynamic per category
  - Default: `"Agentic AI pipeline that threads related news stories, surfaces trends, and delivers daily briefings across Tech, Markets, and Finance."`
- **canonical**: absolute URL тАФ `https://araverus.com/news` or `https://araverus.com/news?category=${category}`
- **openGraph**: title as string (not object), type `website`, image `og-news-default.png` (1200x630), **og:audio** (latest EN briefing `audio_url` when available)
- **twitter**: `summary_large_image`

### JSON-LD тАФ PodcastEpisode
Rendered as `<script type="application/ld+json">` in page component (not metadata). Links to PodcastSeries via `partOfSeries`. Includes `associatedMedia` with audio URL, encoding format, and ISO 8601 duration.

### JSON-LD тАФ PodcastSeries (layout)
Injected in `news/layout.tsx` тАФ applies to all `/news/*` pages. Links show to RSS feed at `/podcast.xml`.

## `/news/[slug]` page (`src/app/news/[slug]/page.tsx`)

### Metadata
- **title**: `item.title` (template appends `| Araverus`)
- **description**: `item.summary` fallback `item.description`
- **canonical**: `https://araverus.com/news/${slug}` (absolute)
- **openGraph**: type `article`, image uses `item.top_image` (no dimensions declared for external images) or `og-news-default.png` fallback (1200x630 declared)
- **twitter**: `summary_large_image`, always

### JSON-LD тАФ NewsArticle + BreadcrumbList

Two schemas per article page:

**NewsArticle:**

| Field | Source | Notes |
|-------|--------|-------|
| `url` | `https://araverus.com/news/${slug}` | Absolute URL |
| `headline` | AI headline from `wsj_llm_analysis.headline` | Never WSJ original title |
| `description` | `wsj_items.description` | RSS original |
| `articleBody` | `wsj_llm_analysis.summary` fallback `description` | AI-generated summary |
| `articleSection` | `feed_name` тЖТ label (TECH тЖТ Tech) | |
| `inLanguage` | `"en"` | |
| `isAccessibleForFree` | `true` (boolean, not string) | No paywall |
| `datePublished` | `wsj_items.published_at` | ISO 8601 |
| `dateModified` | `wsj_items.published_at` | Same as published (articles not edited) |
| `author` | `{ "@type": "Organization", "name": "Araverus" }` | Araverus as curator/publisher |

**Visible Byline**: Article detail page displays `Araverus Team | {date} at {time}` below the headline. Matches JSON-LD author for Google Publisher Center compliance.
| `publisher.logo` | `logo-publisher.png` (600x60) | Separate file from main logo |
| `image` | `item.top_image` (no dimensions) or `og-news-default.png` (1200x630) | External images: no dimensions declared (unknown size) |
| `keywords` | `wsj_llm_analysis.keywords` joined by `, ` | LLM-extracted, 2-4 per article |

**BreadcrumbList:**
```
News (/news) тЖТ Category (/news/c/{slug}) тЖТ Article Title (/news/{slug})
```
All items include `item` (URL) field. Category links to route segment (e.g. `/news/c/tech`), not query param.

### ISR
- `generateStaticParams`: pre-renders latest **100** articles at build
- `revalidate = 7200` (2 hours) тАФ content rarely changes after publish

## Sitemap (`src/app/sitemap.ts`)

| URL | lastModified |
|-----|-------------|
| `/news` | latest article `published_at` |
| `/podcast.xml` | latest article `published_at` |
| `/news/c/*` (x5) | latest article `published_at` |
| `/news/[slug]` (all with headline) | article `published_at` |

- `limit: 10000` тАФ fetches all articles (headline-based visibility gate filters non-ok)
- `changeFrequency` and `priority` removed (Google ignores both)
- `revalidate = 86400` тАФ 24h safety net; on-demand revalidation is primary
- Root `/` excluded (it's a redirect to `/news`)

## Google News Sitemap (`src/app/sitemap-news.xml/route.ts`)

Dedicated News Sitemap for Google News surfaces.

- **Scope**: Last 48 hours only (Google News requirement)
- **Tags**: `<news:publication>`, `<news:publication_date>`, `<news:title>` (AI headline)
- **Cache**: `unstable_cache` 1h revalidation, `s-maxage=3600`
- **Submitted to**: Search Console (one-time)

## Podcast Feed (`/podcast.xml`)

Podcast RSS 2.0 feed with iTunes and Atom namespaces. Exposes the latest EN audio briefing for podcast distribution (Apple Podcasts, Spotify, Google Podcasts). Branded as Araverus.

### Channel
- **title**: `"AI News Briefing тАФ Araverus"`
- **description**: `"Financial intelligence platform powered by AI, machine learning, and neural networks."`
- **itunes:author**: `Araverus`
- **itunes:category**: `News > Daily News`
- **itunes:image**: `podcast-cover.png` (1400x1400 square)
- **itunes:explicit**: `false`
- **itunes:type**: `episodic`
- **atom:link**: self-reference at `/podcast.xml`

### Items
- Single `<item>` with latest EN briefing from `NewsService.getLatestBriefings()`
- `<enclosure>` points to `audio_url` from `wsj_briefings` table
- `<guid>`: `briefing-en-YYYY-MM-DD` (not permalink)
- `<itunes:duration>`: from `audio_duration` column (seconds)

### Cache
- `unstable_cache` with `['news']` tag, 24h revalidation
- `s-maxage=86400, stale-while-revalidate`

### Constraint
Only latest episode exposed (Supabase free tier 1GB). Same URL, content refreshes daily.

## RSS Feed (`/rss.xml`)

News-only feed. Branded as Araverus.

### Channel
- **title**: `"Araverus тАФ Financial Intelligence"`
- **description**: `"AI-powered news threading and financial intelligence briefings across Tech, Markets, and Finance."`
- **language**: `en`
- **copyright**: dynamic year + "Araverus"
- **ttl**: `60` (minutes)
- **lastBuildDate**: latest article `published_at` (not current time)
- **image**: `logo-publisher.png` (no width/height declared)
- **atom:link**: self-reference for feed validators

### Items
- Latest 500 news articles from `NewsService.getNewsItems()` (~7-10 days)
- Each item: title (AI headline), link, guid, pubDate, description, category (feed_name), `<dc:creator>Araverus Team</dc:creator>`
- **Namespace**: `xmlns:dc="http://purl.org/dc/elements/1.1/"` for Dublin Core author attribution
- Only articles with AI headline appear (visibility gate in `getNewsItems`)

### Cache
- `unstable_cache` with `['news']` tag, 24h revalidation
- `s-maxage=86400, stale-while-revalidate`
- SITE fallback: `https://araverus.com` (not localhost)

## robots.ts

**Rule 1 тАФ All crawlers (`*`)**
```
Allow: /
Disallow: /dashboard, /admin
```

**Rule 2 тАФ Googlebot-News**
```
Allow: /news/
```
Path prefix covers `/news`, `/news/[slug]`, `/news?category=*`. Googlebot-News is separate from regular Googlebot тАФ indexes for Google News surfaces (News tab, Discover, Top Stories).

Note: Requires Google News Publisher approval to appear in Google News. This rule removes technical barriers.

## Search Engine Notification (added 2026-03-05)

### WebSub / PubSubHubbub (Google)
- RSS feed (`/rss.xml`) includes `<atom:link rel="hub" href="https://pubsubhubbub.appspot.com/" />`
- Pipeline pings Google's hub after each run via `9_notify_search_engines.py`
- Google re-crawls RSS feed in near-real-time after ping

### IndexNow (Bing, Yandex, Naver)
- API key hosted at `public/{key}.txt`
- Pipeline submits new article URLs in batch via `9_notify_search_engines.py`
- Env var: `INDEXNOW_API_KEY` in `.env.pipeline`
- Endpoint: `https://api.indexnow.org/indexnow` (shared across Bing/Yandex/Naver)
